# Gambit music — drop your track files here

The app plays **one looping track per game** from this folder. Files are loaded
by name; a missing file is simply skipped (no error, no sound), so you can add
them one at a time.

## What to add

Save each track with the **exact filename** below (all lowercase, `.mp3`):

| File | Plays on | Mood to pick |
|------|----------|--------------|
| `lobby.mp3` | Home / menus | Upbeat, welcoming electronic — sets the energy |
| `chess.mp3` | Chess | Cinematic / focus — tense, high-stakes |
| `whot.mp3` | Naija Whot | **Afrobeat / Amapiano** — percussion-driven, on-brand |
| `tictactoe.mp3` | Tic-tac-toe | Light, playful, quirky |
| `snakes.mp3` | Snakes & Ladders | Fun, bouncy, cheerful |
| `blocks.mp3` | Block Blitz | Driving synthwave — keeps the puzzle tempo up |
| `tournament.mp3` | Tournaments | Epic / cinematic — big-moment energy |

Tip: a 1–3 minute track is plenty — it loops seamlessly.

## Where to get free, commercial-OK tracks (no attribution, no licensing risk)

**Pixabay Music** — free for commercial use, no attribution required.
Open a link, pick any track you like, click **Download**, then rename the file
to the name in the table above and drop it in this folder.

- lobby → https://pixabay.com/music/search/upbeat%20electronic/
- chess → https://pixabay.com/music/search/cinematic%20ambient/
- whot → https://pixabay.com/music/search/afrobeat/  (or search "amapiano")
- tictactoe → https://pixabay.com/music/search/playful/
- snakes → https://pixabay.com/music/search/fun%20happy/
- blocks → https://pixabay.com/music/search/synthwave/
- tournament → https://pixabay.com/music/search/epic/

Other good free sources: **Free Music Archive** (filter to CC0 / CC-BY),
**Incompetech** (Kevin MacLeod, CC-BY — needs attribution), **Chosic**.

## After adding files

1. Drop the `.mp3` files in this folder.
2. Commit them and deploy (or just run locally).
3. Turn **Background music** on in Settings — the track switches automatically
   to match whichever game you open.

You don't need all seven to start. Even just `lobby.mp3` + `whot.mp3` already
makes the app feel alive.
